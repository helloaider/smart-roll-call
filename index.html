📋 智能课堂点名器 · 使用说明

✕
🎲 随机点名
⚖️ 均衡抽取 优先从被点次数最少的同学中随机抽取，保证机会均等；本轮全部点完后自动进入下一轮并弹出提示
🌐 全员随机 不考虑被点次数，每次从全班完全随机抽取，适合需要重复点名的场景
🌀 姓名滚动约 1 秒后定格，对应座位同步高亮闪烁
⏳ 滚动期间按钮禁用，等待定格后即可继续点名
✋ 手动点名
👆 双击座位 直接标记该学生为已点名，座位高亮显示，姓名同步更新到顶部
↩️ 再次双击 弹出确认框，确认后取消点名记录，座位恢复未点状态
🖱️ 右键座位 弹出菜单，可查看或编辑该学生的介绍 / 备注
🎨 显示偏好
🔢 被点次数 开启后座位右上角显示累计被点次数，再次点击可隐藏
🏅 班委标记 开启后姓名下方显示彩色下划线：红色 = 班长，绿色 = 组长
🔄 座位视角 教师视角 / 学生视角切换，学生视角下座位表整体左右翻转
☀️ 主题切换 深蓝 / 暗黑两种主题，切换后自动记住偏好
📖 学生介绍
🔛 开启方式 ⚙️ → 显示学生介绍，开启后每次点名自动弹出介绍卡片
📍 显示位置 卡片出现在姓名右侧，约 20 秒后自动消失
✏️ 编辑内容 右键座位 → 查看介绍，点击「编辑」修改，保存后立即生效
⚙️ 点名管理
🗑️ 清空记录 一键清除本轮所有点名记录，⚠️ 操作不可撤销
📤 导出记录 将所有点名数据保存为 Excel 文件，方便存档或换设备使用
📥 导入记录 从之前导出的 Excel 文件恢复数据，支持跨设备迁移历史记录
📊 数据统计
📋 点名统计 查看每位学生的累计被点次数，支持按表头次数排序
🕐 记录明细 按时间顺序查看每次点名详情，同样支持按表头时间排序
💾 统计数据跨会话保存，刷新页面后不丢失
✏️ 编辑座位表 & 配置管理
🖥️ 打开编辑器 ⚙️ → 编辑座位信息，在 builder.html 中可对点名器中的座位及其他信息进行编辑
⚡ 应用配置 编辑完成后点击「↩ 应用到点名器」，无需刷新即时生效
📥 导入 Excel 在编辑器中导入座位表.xlsx 文件或者前期导出的点名器.html文件，批量填入姓名和介绍内容
📤 导出备份 可导出为 Excel 表格，或打包成独立 HTML 文件方便分享和存档
🔁 恢复默认 ⚙️ → 恢复默认配置，将清除全部本地数据，⚠️ 操作不可逆，请提前导出备份
🌐 浏览器配置 可以给页面配置浏览器顶部选项卡显示的标题及icon，页面正文顶部的标题文字等
💾 点名记录、介绍内容、主题及所有偏好设置均自动保存在浏览器本地，关闭或刷新页面后不丢失  ·  🌐 建议使用 Chrome / Edge 等现代浏览器以获得最佳体验
