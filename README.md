# 🎯 智能课堂点名器

一个开箱即用的课堂点名工具，纯原生 HTML 单文件，无需安装任何依赖，双击即可使用。

## ✨ 功能特性

- **随机点名** — 支持「仅抽未点过」和「抽取所有人」两种模式，避免重复或允许重复自由切换
- **手动点名** — 双击座位表中任意学生即可手动标记/取消点名状态
- **座位高亮** — 点名后对应座位实时高亮，全班一目了然
- **视角切换** — 支持教师视角与学生视角（180° 翻转）双向切换
- **双主题** — 内置深蓝 / 暗黑两套主题，切换状态自动保存
- **班委标识** — 红色波浪线标注班长，绿色波浪线标注组长
- **姓名典故** — 点名后可在姓名旁弹出学员介绍卡片，30 秒后自动消失
- **记录管理** — 支持隐藏/显示已点名记录、一键清空
- **数据持久化** — 所有点名记录与设置自动保存至浏览器本地，刷新不丢失
- **粒子背景** — 动态粒子特效，课堂氛围拉满

## 🚀 快速开始

1. 下载 `index.html`
2. 用浏览器打开（推荐 Chrome / Edge）
3. 开始点名

> 无需服务器，无需网络（粒子背景除外），本地直接运行。

## 🛠️ 自定义配置

### 修改学生名单

找到 HTML 文件中的座位表部分，将 `学生01`、`学生02` 等替换为真实姓名：

```html
<td id="张三">张三</td>
<td id="李四" class="class-committee-leader">李四</td>  <!-- 组长 -->
<td id="王五" class="class-committee-monitor">王五</td>  <!-- 班长 -->
```

### 修改班级标题

```html
<div class="chart-header">你的学校-你的教室座位表</div>
```

### 添加姓名典故

在 JS 的 `name_intro` 对象中，以姓名为 key 填写介绍文字：

```js
let name_intro = {
    "张三": "这里填写张三的介绍，点名后将在姓名右侧弹出展示。",
    "李四": "这里填写李四的介绍。"
}
```

### 修改班级口号

```html
<div class="slogan">你的班级专属口号！</div>
```
